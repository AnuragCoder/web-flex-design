A Flexible Layout must have a parent element with the display property set to flex.

Direct child elements(s) of the flexible container automatically becomes flexible items.

When we write display : flex , element comes inline and after that we can use all the flollowing propoeties of Flex

  • Flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-item
  • align-content
